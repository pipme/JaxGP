# Installation Guide

## Using pip

The easiest way to install the most recent stable version of `jaxgp` is
with [pip](https://pip.pypa.io):

```bash
pip install jaxgp
```

## From source

Alternatively, you can get the source:

```bash
git clone https://github.com/pipme/jaxgp.git
cd jaxgp
python -m pip install -e .
```
